Android8.0适配问题

  • Android
  • 适配

  前段时间项目新版本上线，可算松了一口气，可TMD后台bug统计8.0崩溃率100%，你相信嘛？100%啊， 我当时懵逼了！当看完日志以后，心中万条草泥马奔腾！又是该死的适配问题！

Log日志信息

  日志显示的错误信息是Only fullscreen opaque activities can request orientation，直接翻译过来的意思就是只有不透明的activity才能设置方向，8.0以前并没有这样的错误提示。

  网上搜了搜，发现不是个例，这个问题出现在android8.0以上。原因是我们给Activity同时设置了 android:screenOrientation="" 和 <item name="android:windowIsTranslucent">true</item>。

解决方法很简单：
1.找到你设置透明的Activity，然后在他的theme中将android:windowIsTranslucent改为false
<item name="android:windowIsTranslucent">false</item>

2.再加入<item name="android:windowDisablePreview">true</item>就搞定了。
