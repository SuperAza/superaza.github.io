Kotlin学习笔记（一）

Kotlin
  • 基础
  • 语言

  自从Kotlin被Google声明作为Android的官方开发语言以来，Kotlin备受关注，学习的人群也越来越多，Android和Kotlin混合开发的项目也越来越多，各大互联网公司如Facebook、Square等包括国内的BAT，对自己的开源项目也增加了Kotlin版本，作为一名Android狗，没理由不学习下

前言

这是一篇持续更新的博客，会不定期更新学习的内容，记录总结，哎！没办法，脑子笨！看过之后就忘！

基础语法

  如果有语言基础的人看kotlin的语法会比较容易理解，一看就能理解是什么意思，只是书写习惯和平时不太一样，尤其是java，但这并不是重点，多写自然就熟练了，我用的IDE是Intellij IDEA,有许多IDE都可以支持kotlin语言，我就不赘述了

基本数据类型的声明

  Kotlin在定义基本数据类型变量是可以简写var a = 10，甚至连分号都不用写，这属于隐式的定义数据类型，计算机会自动识别出该变量属于什么数据类型，譬如说上边变量a，有可能是Byte类型，也有可能是Int类型，具体是什么类型就看变量a所在方法是需要什么类型.

  变量的定义也可以显示的指定一个类型,成为显式定义，譬如说var a:Byte = 10，这种写法就是将a变量定义为Byte类型，我们知道Byte类型的取值范围是-128~127，如果我们这个时候将a重新赋值 a = 1000,这个时候系统就会报错，因为已经定义了a的数据类型是Byte，1000已经超过了Byte的取值范围，所以报错。

  除了变量的定义，在日常开发中我们还经常使用常量，如上个例子的var HOST_URL = xxx，常量一旦被定义赋值就不可以被更改，这个和Java中的final关键字类似，已经申请了一块内存空间并赋值后便不可进行更改
